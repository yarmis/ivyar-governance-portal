# IVYAR Smart Contracts

Blockchain infrastructure for transparent veteran land allocation.

## Contracts

1. **LandParcelRegistry.sol** - Land parcel registry
2. **VeteranLandGrant.sol** - Veteran grant management  
3. **AntiCorruptionOracle.sol** - AI-powered fraud detection

## Setup
```bash
npm install
cp .env.example .env
# Add PRIVATE_KEY
```

## Deploy
```bash
npm run compile
npm run deploy:mumbai
```
