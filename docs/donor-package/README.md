## üéØ USAGE SCENARIOS

### For Initial Outreach:
1. Start with **Cover Letter** (donor-specific)
2. Include **Executive Summary** (1 page)
3. Attach **Elevator Pitch** (30 seconds)

1. Use **Donor Presentation** (12-15 slides)
